# Ocean Properties - Manual de Identidad para IA

Este documento es la referencia oficial para Claude, Cursor y otros LLMs.

## 1. Paleta de Colores
- **Negro Principal:** `#000000` (Uso para texto y logos sobre blanco)
- **Blanco Principal:** `#FFFFFF` (Uso para fondos y logos invertidos)

## 2. Uso de Logos (Lógica de Fondo)
- **Fondo Claro (Light Mode):** Usar `logos/logo-horizontal-black.png` o `logos/logo-location-black.png`.
- **Fondo Oscuro (Dark Mode):** Usar `logos/logo-horizontal-white.png` o `logos/logo-location-white.png`.
- **Espacios Reducidos / Avatares:** Usar `logos/symbol-black.png` o `logos/symbol-white.png`.

## 3. Tipografía y Estilo
- **Estilo:** Minimalista, limpio, moderno.
- **Marca:** "Ocean Properties Aparthotel & Villas".
